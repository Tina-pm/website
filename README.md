# Tina.pm website

Tina.pm website is developed with [Hugo](https://gohugo.io/).

## Requirements

Install `hugo`. See <https://gohugo.io/installation/>.

In Debian:

```sh
apt install hugo
```

`hugo` version should be at least 0.131.0

## Development

Get environment:

```sh
git clone --recurse-submodules git@github.com:Tina-pm/website.git
cd website
npm install
```

Run local server:

```sh
hugo server
```
