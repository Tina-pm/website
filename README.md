Tina.pm website developed with [Hugo](https://gohugo.io/).

# Requirements

Install `hugo`. See https://gohugo.io/installation/.

In Debian:
```
apt install hugo
```
`hugo` version should be at least 0.131.0

# Development

Get environment:
```
git clone --recurse-submodules git@github.com:Tina-pm/website.git
cd website
npm install
```

Run local server:
```
hugo server
```
