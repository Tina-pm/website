Implementation and integration of translation system for public service information
website.
* Automation of website building and deployment with GitLab pipelines.
* Deployment of a virtual machine management system to reduce number of bare-metal
servers.
* Design and implementation of a complex back-office system providing email, user
authentication, data storage, document management, etc.

**Main technologies:** postfix, dovecot, openldap, samba, libvirt/qemu, weblate, po4a, metalsmith,
bootstrap, sass
