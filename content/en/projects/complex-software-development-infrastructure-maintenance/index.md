Development of various distributed systems communicating via REST APIs, as part of a
complex pipeline:
* Highly-performant rule-based rate-limiter
* Fully audited (blockchain based) user privacy preferences database
* Load-balancing system
* Development and management of a massive distributed fleet of browser-controlling bots
to test user interaction with complex web applications.
* Training and implementation of modern software methodologies: design-document driven
development, release management, reliability engineering, code reviews, automated
code quality testing, process automation, performance instrumentation, etc.
* Deployment of a VPN connecting hundreds of isolated servers and a fully remote
workforce.
* Set-up of a monitoring and log collection stack with automated alerting and deep
instrumentation of applications.
* Development of an automatic testing and building system for continuous delivery.
* Automation of infrastructure management

**Main technologies:** ansible, prometheus, grafana, loki, memcached, redis, mysql, python,
nodeJS, perl, flask, puppeteer
